¡Perfecto! Esto confirma que tu página web está funcionando correctamente, pero el servidor no estaba
 configurado para cargar automáticamente el archivo index.html como página principal.

Para solucionar esto de forma definitiva:

Crea o edita el archivo .htaccess:

En el directorio raíz (htdocs), asegúrate de tener un archivo llamado .htaccess.
1- Si no existe, créalo y agrega el siguiente contenido:
DirectoryIndex index.html
Guarda el archivo como .htaccess (sin nombre, solo extensión). Si usas Notepad, asegúrate de seleccionar 
"Todos los archivos"  en el tipo de archivo al guardar, para que no lo guarde como .htaccess.txt.

2-Usa FileZilla o el administrador de archivos de tu hosting para subir el archivo:

Conéctate al servidor usando FileZilla.
Sube el archivo .htaccess al directorio htdocs.
